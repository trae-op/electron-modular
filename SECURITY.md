# Security Policy

## Supported Versions

We support the latest published minor version on the default branch. Please update to the newest release before reporting.

## Reporting a Vulnerability

Please report security vulnerabilities privately by emailing: security@devisfuture.com.

Include as much detail as possible:

- Description of the issue
- Steps to reproduce
- Impact assessment
- Affected versions
- Proof-of-concept (if available)

We will acknowledge receipt within 72 hours and provide a timeline after initial triage.

## Disclosure Policy

We follow responsible disclosure practices. Please do not file public issues for security vulnerabilities.
